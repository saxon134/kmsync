# 产品需求文档

## 1. 产品定位

本产品是一个跨设备输入与剪贴板共享工具，支持 macOS、Windows、Linux。用户在一台主控电脑上使用鼠标、键盘和剪贴板时，可以无缝控制同一账户下绑定的其他电脑。

产品体验目标是：多台电脑仍然保留各自系统环境，但输入体验像一台电脑的多屏扩展。

## 2. 目标用户

- 同时使用 Mac 和 Windows / Linux 的开发者、设计师、运营、数据分析人员。
- 有多台电脑但不希望频繁切换键盘鼠标的办公用户。
- 使用台式机 + 笔记本 + 工作站组合的专业用户。
- 对低延迟、高稳定性和操作习惯一致性敏感的高级用户。

## 3. 核心价值

1. 减少多设备之间切换键鼠的成本。
2. 在不同系统之间保持用户熟悉的快捷键与滚轮习惯。
3. 同步复制/粘贴内容，降低跨设备转移文本、链接、图片和文件的摩擦。
4. 通过账户体系管理多台设备，避免手动配置 IP。
5. 局域网低延迟直连，提供接近本机的输入体验。

## 4. 核心场景

### 4.1 Mac 控制 Windows

用户在 MacBook 上移动鼠标到屏幕右侧，光标进入 Windows 台式机。用户按下 `Command + C`，远端 Windows 应收到 `Ctrl + C`。滚轮方向可以保持 Mac 用户习惯。

### 4.2 Windows 控制 Mac

用户在 Windows 电脑上控制 Mac。用户可以设置 `Ctrl` 映射为 `Command`，从而让 `Ctrl + C` 在 Mac 上执行复制。

### 4.3 多设备剪贴板同步

用户在一台电脑复制文本、链接、图片或文件，在另一台电脑直接粘贴。文本剪贴板作为 MVP，图片和文件剪贴板后续扩展。

### 4.4 设备 IP 自动更新

用户无需记住设备 IP。客户端启动、网络变化、唤醒或定时心跳时自动向服务端上报局域网 IP、公网 IP、端口、NAT 信息和在线状态。

### 4.5 跨系统操作习惯同步

用户可以选择“保持 Mac 习惯”“保持 Windows 习惯”“按目标系统习惯”或“自定义”。系统自动生成默认键位映射、快捷键映射、滚轮方向和鼠标参数。

## 5. 功能范围

### 5.1 账户与设备

- 邮箱/手机号登录。
- 可选支持 Apple / Google / Microsoft OAuth。
- 一个账户绑定多台电脑。
- 每台电脑拥有唯一 `device_id` 和设备密钥对。
- 支持设备命名、设备类型、系统版本、客户端版本、在线状态展示。
- 支持设备解绑、重命名、禁用、重新授权。

### 5.2 鼠标共享

- 支持鼠标跨屏切换。
- 支持多显示器布局。
- 支持目标设备边缘方向配置。
- 支持滚轮同步。
- 支持滚轮方向反转。
- 支持水平滚轮。
- 支持鼠标速度倍率和加速度策略。
- 支持热键锁定当前设备。

### 5.3 键盘共享

- 支持普通按键、功能键、方向键、数字键盘。
- 支持修饰键映射：
  - Command
  - Control
  - Option / Alt
  - Windows / Super
  - Shift
  - Caps Lock
- 支持左右修饰键分别映射。
- 支持物理键位模式和字符语义模式。
- 支持快捷键语义映射，例如复制、粘贴、撤销、查找、全选、应用切换。

### 5.4 剪贴板共享

MVP 支持：

- 纯文本。
- URL。
- 基本文本格式保留。

后续支持：

- 图片。
- 富文本。
- 文件复制/粘贴。
- 剪贴板历史。
- 敏感应用过滤。
- 自动过期。

### 5.5 习惯配置

预设：

- 保持 Mac 习惯。
- 保持 Windows 习惯。
- 保持 Linux 习惯。
- 按目标系统习惯。
- 完全自定义。

高级配置：

- Command / Ctrl / Windows / Alt 互换。
- 左右修饰键单独映射。
- 滚轮方向设置。
- 水平滚轮方向设置。
- 鼠标速度倍率。
- 加速度策略。
- 键盘布局。
- 目标设备专属配置。
- 应用级例外规则。

### 5.6 连接与发现

- 局域网自动发现。
- 同账户设备列表自动同步。
- LAN 直连优先。
- P2P NAT 穿透。
- 无法直连时使用 Relay。
- 网络变化自动重连。
- 设备休眠/唤醒后自动恢复。

## 6. 非目标

MVP 不做完整远程桌面。

MVP 不传输屏幕画面。

MVP 不保证 Linux Wayland 下所有桌面环境完整支持全局输入捕获和注入。

MVP 不处理复杂远程输入法状态同步。

MVP 不支持 Windows 登录界面和 UAC 安全桌面输入注入，后续通过系统服务扩展。

## 7. 产品信息架构

客户端主要页面：

- 设备列表
- 设备布局
- 当前连接状态
- 操作习惯设置
- 剪贴板同步设置
- 网络与连接诊断
- 权限引导
- 账户与安全
- 自动更新

## 8. 成功指标

体验指标：

- 局域网鼠标端到端延迟小于 5ms 到 10ms。
- 键盘事件端到端延迟小于 10ms。
- 切换目标设备感知延迟小于 50ms。
- 断线自动重连小于 1s 到 3s。
- 空闲 CPU 占用小于 1%。

产品指标：

- 新用户 5 分钟内完成两台设备连接。
- 权限引导完成率高于 80%。
- 首次连接成功率高于 90%。
- 一周留存中，至少 50% 用户启用剪贴板同步或键位映射。
